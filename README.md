
# CESINF

Site vitrine présentant les activités CESINF :
analyse des besoins métiers, organisation des outils,
automatisation de tâches et intégration de solutions logicielles adaptées aux PME.

---

## Objectif du projet

Ce site a pour objectif de présenter :

- la mission CESINF,
- les domaines d’expertise,
- les solutions proposées,
- une synthèse des approches possibles,
- les moyens de contact.

Le site a été conçu comme une page unique avec navigation fluide entre sections.

---

- CSS3
- JavaScript

- GitHub
